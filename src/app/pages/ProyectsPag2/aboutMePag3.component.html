Expercia Laboral - GIF Generator

Desliza para ver más

Como prueba de ingreso para Puropollo, desarrollé una aplicación web utilizando Angular con el objetivo de demostrar mis habilidades en la construcción de soluciones eficientes, escalables y centradas en la experiencia del usuario. Implementé una experiencia de navegación continua, incorporando memoria de scroll, lo que permite que, al actualizar o volver a cargar la página, el usuario retome automáticamente desde la misma posición en la que se encontraba. Esta funcionalidad mejora significativamente la usabilidad, especialmente en listas largas de contenido.

Además, desarrollé un sistema de historial de búsqueda local, almacenado en localStorage, que permite a los usuarios acceder rápidamente a sus consultas anteriores. Esta característica mejora la accesibilidad a la información y reduce el tiempo de navegación. La aplicación está organizada bajo una arquitectura modular, con componentes reutilizables y servicios inyectables, lo que facilita su escalabilidad y mantenimiento. Incorporé lazy loading para optimizar el rendimiento, cargando módulos solo cuando el usuario los necesita, y apliqué diseño responsive para garantizar una visualización correcta en distintos dispositivos.

Durante el desarrollo, enfrenté y resolví conflictos relacionados con errores de versiones y dependencias en Angular, que requirieron una reconfiguración manual del entorno y una reestructuración parcial del proyecto. Esto me permitió demostrar capacidad de análisis, resolución de problemas y adaptación a contextos técnicos complejos. Finalmente, prioricé la organización limpia del código, la estabilidad de la aplicación y una interfaz intuitiva, enfocados en brindar una experiencia fluida y profesional, acorde a los estándares de calidad que busca una empresa como Puropollo.

Proyecto Personal - Buscador de Países

Desliza para ver más

Desarrollé una aplicación web interactiva que permite a los usuarios buscar información sobre países por su nombre, capital o región, utilizando la API REST Countries para obtener datos en tiempo real como bandera, capital, población y región. El proyecto fue construido con Angular 19, aprovechando sus nuevas características como los signals y observables para una gestión reactiva del estado.

Implementé una arquitectura modular con lazy loading y componentes desacoplados para mejorar el rendimiento y la escalabilidad. Utilicé servicios inyectables para centralizar la lógica de acceso a datos y comunicación entre componentes. Para mejorar la experiencia del usuario, incorporé un sistema de memory scroll y persistencia en localStorage, lo que permite retomar la navegación desde el mismo punto incluso tras cerrar o recargar la página.

Este proyecto representa la aplicación práctica de técnicas modernas en desarrollo frontend, con un fuerte enfoque en la optimización del rendimiento, la experiencia del usuario y la organización eficiente del código.

Expercia Laboral - Teslop Shop

Desliza para ver más

Desarrollé en un equipo de 5 personas una aplicación web e-commerce utilizando Angular 19 como parte de un emprendimiento de venta de ropa para niños, jóvenes y adultos, con el objetivo de introducir la marca en el mundo del marketing digital y ofrecer una plataforma sólida y escalable para ventas en línea. La aplicación presenta un diseño moderno y responsive, adaptado a todo tipo de dispositivos. Implementamos un sistema completo de autenticación, carrito de compras y proceso de checkout funcional. Para el manejo de pagos en línea, integramos Stripe, permitiendo el procesamiento seguro y eficiente de transacciones, junto con un sistema de inventario en tiempo real.

Utilizamos Firebase como backend, donde gestionamos la autenticación de usuarios, el almacenamiento de productos y el seguimiento de pedidos. También incorporamos lazy loading para mejorar el rendimiento general, reducir los tiempos de carga y optimizar la experiencia de navegación. La aplicación incluye un panel de administración robusto, desde el cual se pueden gestionar productos, categorías y pedidos, brindando control total sobre el negocio.

Durante el desarrollo, enfrenté y resolví desafíos relacionados con la optimización de imágenes, el manejo de estados globales y la sincronización de datos en tiempo real, garantizando una experiencia de usuario fluida y profesional. Esta solución está pensada para posicionarse estratégicamente en el entorno digital y facilitar el crecimiento del emprendimiento mediante canales online.
