GIF Generator

Desliza para ver más

Desarrollé una aplicación web utilizando Angular, orientada a brindar una experiencia de navegación continua y eficiente. La interfaz cuenta con memoria de scroll, lo que permite que, al recargar la página, el usuario retome su navegación exactamente desde la posición en la que se encontraba. También implementé un sistema de historial de búsqueda para mejorar la accesibilidad a consultas previas, facilitando la interacción y reduciendo el tiempo de navegación.

La aplicación está estructurada con una arquitectura modular y componentes reutilizables, lo que facilita su escalabilidad y mantenimiento. Incorporé diseño responsive para asegurar una correcta visualización en distintos dispositivos, y utilicé técnicas de optimización como lazy loading para mejorar el rendimiento general.

Durante el proceso, resolví conflictos derivados de errores en la configuración del entorno y en las versiones de Angular, lo cual implicó reestructurar parte del proyecto y ajustar manualmente ciertas dependencias. Además, presté especial atención a la organización del código y a la experiencia del usuario final, priorizando la eficiencia en la navegación, la estabilidad del sistema y la claridad en la interacción.

Buscador de Países

Desliza para ver más

Desarrollé una aplicación web interactiva que permite a los usuarios buscar información sobre países por su nombre, capital o región, utilizando la API REST Countries para obtener datos en tiempo real como bandera, capital, población y región. El proyecto fue construido con Angular 19, aprovechando sus nuevas características como los signals y observables para una gestión reactiva del estado.

Implementé una arquitectura modular con lazy loading y componentes desacoplados para mejorar el rendimiento y la escalabilidad. Utilicé servicios inyectables para centralizar la lógica de acceso a datos y comunicación entre componentes. Para mejorar la experiencia del usuario, incorporé un sistema de memory scroll y persistencia en localStorage, lo que permite retomar la navegación desde el mismo punto incluso tras cerrar o recargar la página.

Este proyecto representa la aplicación práctica de técnicas modernas en desarrollo frontend, con un fuerte enfoque en la optimización del rendimiento, la experiencia del usuario y la organización eficiente del código.

Teslop Shop

Desliza para ver más

Desarrollé una aplicación web de comercio electrónico utilizando Angular, que permite a los usuarios explorar productos, agregarlos al carrito, autenticarse, y realizar operaciones básicas de compra. La interfaz fue diseñada con un enfoque responsive, combinando Bootstrap para algunos componentes específicos y TailwindCSS para lograr una apariencia moderna, flexible y coherente.

El proyecto incluye un sistema de autenticación de usuarios, gestión dinámica de productos y actualización en tiempo real del carrito de compras. La aplicación fue desplegada en Netlify para facilitar su acceso en línea.

El frontend se conecta con un backend desarrollado de forma independiente y ejecutado en un contenedor Docker, desde donde se obtiene toda la información de productos, usuarios y operaciones. Esta separación entre frontend y backend permitió una arquitectura escalable y modular. La integración entre ambos servicios se realizó mediante peticiones HTTP a una API personalizada.
